# Shopee AI 描述生成器 (30功能版) - 完整教學文件

**版本：v2.4 (全部 30 功能一次完成)**  
**目標用戶**：蝦皮賣家、電商經營者  
**主要功能**：使用 Claude / Gemini AI 快速生成高轉換率商品描述、廣告文案、SEO 優化、利潤分析、批量處理等 30 種實用工具。

---

## 1. 安裝與載入擴充功能

### 步驟：
1. 下載或 clone 專案：
   ```bash
   git clone https://github.com/netfox-web/shopee-ai-desc-generator.git
   cd shopee-ai-desc-generator
   ```

2. 開啟 Chrome / Edge 擴充功能頁面：
   - 輸入網址：`chrome://extensions/` 或 `edge://extensions/`

3. 開啟右上角「開發人員模式」

4. 點擊「載入未封裝項目」

5. 選擇專案資料夾（包含 `manifest.json` 的資料夾）

6. 擴充功能圖示會出現在工具列（可固定）。

**快捷鍵**：預設 `Ctrl + Shift + A`（可自訂）

---

## 2. 初始設定（必做）

點擊擴充功能圖示 → 點「設定」按鈕，或直接開啟 `chrome://extensions/` → 找到擴充功能 → 「詳細資料」→ 「擴充功能選項」。

### 必要欄位：
- **Claude API Key**（推薦，主要使用）：從 https://console.anthropic.com/ 取得 `sk-ant-...`
- **Gemini API Key**（備用）：從 https://makersuite.google.com/app/apikey 取得 `AIza...`

至少填寫一個即可使用全部功能。系統會智能切換（優先 Claude）。

### Pro 模式（解鎖進階功能 26-30）：
- 勾選「啟用 Pro 會員（本地模擬）」
- 這是本地模擬，實際上線時可接付費系統。

儲存後即可使用。

---

## 3. 基本使用方式

### 三種主要入口：
1. **Popup（彈出視窗）**：點擊工具列圖示
2. **Side Panel（側邊欄，推薦）**：點擊 popup 裡的「開啟完整 Side Panel」或長按圖示
3. **頁面內按鈕**：進入蝦皮商品編輯頁，會自動出現「🤖 AI 30功能生成」按鈕

### 使用流程（最常見）：
1. 打開蝦皮「編輯商品」頁面（或任何商品頁）
2. 點擊擴充功能圖示
3. 勾選想要的功能（可多選做批量）
4. 點「生成描述」
5. AI 自動產生內容 → 自動複製到剪貼簿 + 自動填入頁面文字框

**右鍵快速生成**（功能 17）：在頁面任意處右鍵 → 選擇「使用 Shopee AI 生成描述」

## 4. 30 個功能完整說明

功能已分類，Popup 與 Side Panel 會依類別顯示，並標註狀態。

### 內容生成類（1-7）
- **1. 多模板選擇**：描述 / 廣告 / 規格 / SEO / 促銷 / 評價 / 利潤計算（已實作核心）
- **2. 廣告文案一鍵生成**：適合 FB / 首頁廣告，強調 urgency
- **4. SEO 標題 + 關鍵字優化**：一次產生多個 SEO 標題 + 描述（已實作）
- **5. 多語言翻譯**：中英越泰自動翻譯，保留銷售語氣
- **6. 限時優惠 / 促銷文案**：倒數計時、折扣強調
- **7. 評價回覆自動生成**：5 種不同風格回覆模板

### 定價與競爭（8-12）
- **8. 同類商品價格監控**：啟動後每小時自動檢查 + AI 建議
- **9. AI 建議售價**：根據成本、競品、目標利潤給 3 種方案
- **10. 降價 / 調價提醒**：結合銷售趨勢給建議
- **11. 利潤計算器**（已實作加強版）：輸入成本、運費、平台費 → 即時算售價與 ROI
- **12. 競品分析報告**：比較規格、價格、評價，給優勢/改進建議

### 自動化（13-17）
- **13. 批量商品描述生成**：一次處理多個商品（支援列表頁勾選）
- **14. 一鍵搬家工具**：PChome / MOMO / Shopify → Shopee 格式自動轉換
- **15. 圖片 Alt Text 自動生成**：SEO + 可訪問性優化
- **16. 商品圖片優化建議**：構圖、燈光、角度建議
- **17. 右鍵快速生成**：選取文字或圖片即可觸發

### 數據分析（18-21）
- **18. 銷售數據 AI 洞察**：熱門時段、暢銷款式、改進方向
- **19. 熱門關鍵字追蹤**：推薦 10 個高潛力關鍵字
- **20. 退貨 / 客訴分析**：分類問題並給預防建議
- **21. 活動檔期建議**：雙11、618、母親節等專屬策略

### 顧客與訂單（22-25）
- **22. 買家評價風險預警**：負評趨勢偵測 + 改善文案
- **23. 客服回覆模板庫**：詢價、退貨、延遲等常見情境
- **24. 訂單優先處理建議**：高價值、急單、問題單排序
- **25. 自動發送追蹤訊息**：出貨後 3 天 / 7 天 / 到貨後模板

### 進階與變現（26-30，Pro 功能）
- **26. Prompt 模板自訂**：在設定頁自訂或覆蓋提示詞
- **27. 團隊共享模板**：上傳 / 下載團隊常用模板（模擬）
- **28. 使用量統計儀表板**：本月生成次數、熱門功能、成本估計
- **29. Pro 會員功能鎖定**：開啟 Pro 後解鎖 26-30（本地模擬）
- **30. 多平台同步**：Shopee ↔ PChome / MOMO / Shopify 格式轉換

## 5. 進階操作教學

### 批量生成（功能 13）
1. 進入蝦皮「我的商品」列表頁
2. 勾選多個商品（或直接點「批量生成」按鈕）
3. 在 popup 勾選想要的模板
4. 點擊生成 → 系統會逐一產生並可一次複製或下載 CSV

### 價格監控（功能 8、10）
- 在 popup 點「啟動價格監控」
- 每小時自動背景檢查（使用 Chrome alarms）
- 發現異常會跳通知 + AI 給降價建議
- 可在 Side Panel 查看最新報告

### 一鍵搬家工具（功能 14）
1. 點擊 popup 的「一鍵搬家」
2. 選擇來源平台
3. 貼上其他平台的商品 JSON 或文字描述
4. AI 自動轉成適合蝦皮的標題 + 描述 + 規格

## 6. 常見問題 FAQ

**Q: 沒有 API Key 可以用嗎？**  
A: 可以，但會使用本地模擬描述（功能仍完整，只是品質較低）。建議至少填一個金鑰。

**Q: 生成的內容會直接覆蓋嗎？**  
A: 不會。會自動填入 + 複製，你可以先預覽再決定是否保留。

**Q: 支援哪些蝦皮頁面？**  
A: 商品編輯頁、商品列表頁、商品詳情頁。其他頁面右鍵仍可使用。

**Q: 如何切換 Claude / Gemini？**  
A: 在設定頁選擇「預設偏好模型」，或兩個都填，系統會自動選擇。

**Q: 批量生成會不會很慢？**  
A: 視 AI 回應速度而定（通常 3-8 秒一個）。建議一次不要超過 20 個。

**Q: 價格監控會不會一直跑？**  
A: 只在你手動啟動後才會背景執行，可隨時在 Side Panel 關閉。

**Q: 想自訂提示詞怎麼辦？**  
A: 功能 26「Prompt 模板自訂」已支援，在設定頁可編輯常用模板。

## 7. 開發者 / 進階自訂

- 所有 Prompt 集中在 `background.js` 的 `FEATURE_PROMPTS` 物件
- 想加新功能：新增 id → 在 FEATURES 陣列 + Prompt → 更新 UI
- 真實 API 呼叫位於 `callClaude` / `callGemini`
- 監控使用 `chrome.alarms`
- 右鍵選單在 `background.js` 的 `chrome.contextMenus`
